# Tool/Product Mention Guidelines (X.com Edition)

Only mention tools if user requests it. Default is NO tool mentions.

---

## Key Difference from Reddit

On Reddit, tool mentions hide in paragraphs. On X.com, every word is visible. A 280-character tweet with a tool mention IS the tool mention. Subtlety is harder and more important.

---

## Subtlety Levels

### Level 1-2 (Barely There)
"my friend set up some automation thing that saves me like 20 min a day. no idea how it works honestly"
- Tool not named
- Extremely casual
- Barely relevant to the tweet's point

### Level 3-4 (Brief Mention)
"been using [tool] for a few weeks. jury's still out but it did catch something I would've missed"
- Named once, lowercase
- Hedged immediately
- Small specific benefit, not transformative

### Level 5-6 (Noticeable but Not Dominant)
"tried [tool] for [specific task]. it's not perfect -- breaks on [specific thing] -- but honestly it's faster than doing it manually for the basic stuff"
- More detail on use case
- Specific limitation included
- Not positioned as solution to everything

### Level 7-10 (Don't Use -- Too Promotional)
Will read as an ad and get muted/blocked/ratio'd.

---

## Placement Rules (Threads)

### Where to Put It:
- **Middle tweets** (not hook, not closer)
- **After establishing the real insight** (the human discovery, not the tool)
- **As a detail in a larger story**, not the point of the story

### Where NOT to Put It:
- Hook tweet (instant ad detection)
- As the turning point ("then I found [tool]...")
- Final tweet (sounds like CTA)
- Multiple tweets (repetition = sponsored content)

## Placement Rules (Single Tweets)

A single tweet that mentions a tool IS a promotional tweet. Only viable if:
- The tweet is genuinely about your experience, tool is incidental
- The tone includes doubt or limitation
- It would make sense even if the tool name were removed

---

## Language Patterns

### Phrases That Work:
- "been messing around with [tool]"
- "my friend got me into [tool], still figuring it out"
- "tried [tool] for this. it mostly works"
- "not sure if [tool] is good or if I'm just bad at the alternative"

### Phrases That DON'T Work:
- "I discovered [tool] and it's a game-changer"
- "[tool] is the best tool for X"
- "if you're not using [tool] you're behind"
- "check out [tool]" (direct promo)
- "shoutout to [tool]" (transparent promo)

---

## Required Disclaimers

Include at least one:
- "still figuring it out"
- "might just be me but"
- "it breaks sometimes"
- "not perfect"
- "idk if it's actually good or if the bar is just low"

---

## Promotion Test

**Critical:** Remove the tool mention entirely. Does the tweet/thread still make sense and have value? If no, it's an ad. Rewrite from scratch with the story first, tool mention optional.

---

## X.com-Specific Risks

- **Shadowban risk:** Repeated tool mentions across tweets can trigger spam detection
- **Ratio risk:** Obvious promotion gets quote-tweeted mockingly
- **Mute risk:** Followers mute accounts that feel like ad channels
- **Trust erosion:** One obviously sponsored tweet makes every future mention suspect

The safest approach: make the tool mention so casual that nobody screenshots it as an example of "this person is clearly being paid to post this."
